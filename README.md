## ✨ Особенности
- 📝 Создание и редактирование задач с описанием
- ✅ Отметка выполнения (чекбокс, зачёркивание, прозрачность)
- 🎨 Настройка акцентного цвета (палитра + HEX)
- 🌓 Тёмная и светлая темы
- 🔤 Поддержка пользовательских шрифтов (TTF, OTF, WOFF)
- 📏 Регулировка размера текста для заголовка и описания
- 📱 Полностью адаптивный дизайн
- 💾 Автосохранение в localStorage

## 🛠 Использование

1. Откройте [веб-приложение](https://somenmi.github.io/mny/)
2. Добавляйте задачи в поле ввода
3. Отмечайте выполненные
4. Используйте кнопки редактирования/удаления
5. Настройте внешний вид в меню настроек под себя

## ⌨️ Горячие клавиши
- `Enter` - добавить задачу (в поле ввода)
- `Esc` - закрыть модальные окна

## 💾 Хранение данных
Все данные сохраняются в `localStorage` вашего браузера:
- Задачи, описание и их статус
- Выбранная тема
- Акцентный цвет
- Загруженные шрифты
- Размеры текста
- Настройки отображения
